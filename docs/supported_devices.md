# Supported devices

## Supported
- ThesslaGreen AirPack Home series compatible with manufacturer Modbus protocol.
- AirPack variants with Modbus TCP.
- AirPack variants with RTU/USB, if configured and supported by the installation.

## Partially supported
- Devices/firmwares where only part of the Modbus register map is available.

## Unsupported / unknown
- Devices without Modbus enabled.
- Devices with unsupported firmware/register layout.
- Devices where another controller keeps exclusive Modbus connection.

Available functions depend on model, firmware and detected capabilities.
